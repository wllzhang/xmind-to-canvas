# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-21

### Added
- Initial release
- XMind file parsing using official XMind SDK
- Automatic layout calculation with ELK.js
- JSON Canvas format generation
- Command palette integration
- Right-click context menu for .xmind files
- Support for multiple XMind versions (legacy, Zen, 2020+)

### Features
- Convert XMind mind maps to Obsidian Canvas
- Professional graph layout engine
- Hierarchical tree layout for mind maps
- Automatic node positioning and spacing
- Edge connections between parent and child nodes

### Technical
- TypeScript implementation
- Rollup build system
- ESLint code quality checks
- GitHub Actions CI/CD pipeline
